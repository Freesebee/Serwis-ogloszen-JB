Projekt zaliczający przedmiot:
Programowanie aplikacji biznesowych w oparciu o platformę Java (2022)
Pracownia Specjalistyczna 3

Autorzy projektu:
- Konrad Augustynowicz
- Jakub Aleksiejuk
- Maciej Kuczyński

Temat projektu: Serwis ogłoszeń

Wymagania:
- wersja Java: 11(LTS)
- Serwer bazodanowy: nie dotyczy(baza na heroku)
- NodeJS

Lista funkcjonalności:
- Podział na role użytkowników (3 minimum),
- Testy jednostkowe,
- Dziennik zdarzeń,
- Dane przechowywane w relacyjnej bazie,
- Uwierzytelnianie,
- Informowanie użytkownika o zaakceptowaniu, wygaśnięciu i publikacji ogłoszenia poprzez e-mail,
- Moderowanie ogłoszeń przed ich publikacją,
- Przeglądanie/dodawanie/edycja/usuwanie kategorii,
- Przeglądanie/dodawanie/edycja/usuwanie ogłoszeń.

Instrukcja zbudowania:
- Aplikacja kliencka (Angular):
Należy zainstalować niezbędne pakiety poprzez użycie komendy 'npm ci', a następnie 
uruchomić aplikację poprzez 'npm run start'.

- WebAPI (Spring Boot):
Projekt należy skompilować i uruchomić
